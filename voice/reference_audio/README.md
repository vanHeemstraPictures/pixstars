# Reference Audio

Place your authorized reference WAV files here.

## Purpose

These files are used by the Coqui XTTS backend to create and cache a reusable speaker identity for the Pixstars lamp voice.

## Recommended characteristics

- clean speech
- low background noise
- multiple short clips are fine
- mono or stereo WAV is acceptable
- English clips if your output language is English

## Important

Use only material you have the right to use. This repository is designed for an **E.T.-inspired** character voice, not deceptive impersonation.
