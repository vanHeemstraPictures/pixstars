# Knowledge - Pixar

This folder is the lamp's RAG knowledge base about Pixar - specifically
the studio's early years, its short films, and the lineage from John
Lasseter's "Luxo Jr." (1986) forward. The Pixstars lamp is a spiritual
cousin of Luxo, and the audience will read that reference whether the
script names it or not.

## What goes here

Plain markdown files, one topic per file. Examples (create as needed):

- luxo_jr.md         - the 1986 short, why a desk lamp became a character
- pixar_history.md   - studio timeline, key shorts, founding figures
- character_design.md - notes on how Pixar gives objects personality
                        without dialogue (relevant to Pixstars)
- silence_in_pixar.md - examples of wordless storytelling (Wall-E opening,
                        Up opening montage)

## What does not go here

- Plot summaries of full features (too long, too specific).
- Modern Pixar / Disney corporate material.
- Anything that would make the lamp sound like a wiki.

## Format

Short, quotable, one idea per file. Straight ASCII quotes only.

## Why this folder matters

Pixstars is performed silently and leans on the same tradition Pixar
codified for animated objects: a desk lamp can carry an arc with no
words. Material here should sharpen the lamp's instinct for that
tradition, not turn it into a film-school lecture.
