# Optional Hivemind Layer

## What this layer is

An optional Mac-side workflow automation layer for the Pixstars project.

## What it is not

It is not the replacement for:
- HiveMind server
- Pi endpoint
- Ardour playback
- real-time lamp logic

## Best role

Treat it as a backstage factory manager for:
- data preparation
- render automation
- evaluation and approval
- repo housekeeping
- asset publication
